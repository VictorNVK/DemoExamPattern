# Входные файлы импорта

Положите сюда файлы демоэкзамена (из Приложения 2):

| Файл | Назначение |
|------|------------|
| `Tovar.xlsx` | Товары (**обязательно**) |
| `user_import.xlsx` | Пользователи |
| `Заказ_import.xlsx` | Заказы |
| `Пункты выдачи_import.xlsx` | Адреса пунктов выдачи |
| `1.jpg`, `2.jpg`, … | Фото товаров |

Также можно положить файлы в `../test_files/` в корне репозитория.

Импорт выполняется **один раз при старте backend**, если все таблицы (`users`, `products`, `orders`) пустые.

После очистки БД перезапустите:

```powershell
cd demo-exam-spring-backend
.\gradlew.bat bootRun
```

В логе должно быть: `Импорт Excel завершён: users=..., products=..., orders=...`
