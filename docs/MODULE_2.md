# Модуль 2. Авторизация и список товаров

Соответствует заданию **М2** (КОД 09.02.07-2-2026): окно входа, роли пользователей, вывод каталога товаров.

## Требования ТЗ

- При запуске — окно авторизации (логин/пароль из БД).
- Возможность войти как **гость** без учётной записи.
- После входа — ФИО и роль в правом верхнем углу, выход на экран входа.
- Список товаров для **гостя, клиента, менеджера, администратора**.
- Отображение: фото (или заглушка `picture.png`), наименование, категория, описание, производитель, поставщик, цена, ед. изм., остаток, скидка.
- Подсветка строк:
  - скидка **> 15%** — фон `#2E8B57`;
  - сниженная цена — старая перечёркнута (красная), итоговая чёрная;
  - **нет на складе** — голубой фон.

## Реализация

### Backend

| Компонент | Путь |
|-----------|------|
| Авторизация | `AuthController`, `AuthService` |
| Токен | `AuthTokenService`, `TokenAuthenticationFilter` |
| Список товаров | `ProductController` → `GET /api/products` |
| Изображения | `GET /api/files/images/{name}` |

**API авторизации:**
```
POST /api/auth/login
Body: { "login": "admin", "password": "admin" }
Response: { "token", "userId", "fullName", "role" }
```

Дальнейшие запросы: заголовок `Authorization: Bearer {token}`.

### Client

| Экран / логика | Файл |
|----------------|------|
| Окно входа | `ui/screens/LoginScreen.kt` |
| Список товаров | `ui/screens/ProductListScreen.kt` |
| Карточка товара | `ui/components/ProductCard.kt` |
| Роли | `model/UserRole.kt` |
| Состояние приложения | `app/AppState.kt` |

### Роли и доступ (UI)

| Роль | Каталог | Поиск/фильтр | Заказы | CRUD товаров |
|------|---------|--------------|--------|--------------|
| Гость | ✓ | — | — | — |
| Client | ✓ | — | — | — |
| Manager | ✓ | ✓ | ✓ | — |
| Admin | ✓ | ✓ | ✓ | ✓ |

Права на backend дублируются в `SecurityConfig` и `@PreAuthorize` на контроллерах.

## Проверка модуля

1. Запустить backend и client.
2. **Гость** — кнопка «Продолжить как гость», каталог без фильтров.
3. **client/client** — вход, каталог без панели поиска.
4. **manager/manager** — каталог + панель поиска/фильтра/сортировки + кнопка «Заказы».
5. **admin/admin** — то же + кнопки добавления/редактирования товаров.
6. Товар со скидкой > 15% и с нулевым остатком — проверить подсветку.
