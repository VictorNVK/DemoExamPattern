## Реализация

### Backend

| Функция | API |
|---------|-----|
| Вход | `POST /api/auth/login` |
| Список товаров | `GET /api/products` |
| Картинки | `GET /api/files/images/{name}` |

Роли в БД: `client`, `manager`, `admin`. Токен Bearer для защищённых методов.

## Алгоритм (кратко, для блок-схемы)

```
Старт
  → Окно входа
  → [Гость] → Каталог (без фильтров)
  → [Логин/пароль] → API login
       → ошибка → сообщение
       → успех → сохранить сессию → Каталог (по роли)
  → Каталог: загрузка GET /api/products
  → Отображение карточек с подсветкой
  → [Выход] → очистка токена → Окно входа
```

Оформите по **ГОСТ 19.701-90** в visio/draw.io → PDF.
