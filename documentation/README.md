## Архитектура проекта

```
Клиент (Compose Desktop)  ──HTTP──►  Backend (Spring Boot)  ──►  SQLite
```

В ТЗ предполагается монолит с локальной БД; в проекте — **клиент + сервер**, что допустимо при сохранении всего функционала через API.
