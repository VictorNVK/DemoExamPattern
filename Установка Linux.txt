ИНСТРУКЦИЯ ПО УСТАНОВКЕ: Continue + Ollama на Linux

Данная инструкция поможет развернуть полностью локального AI-ассистента для разработки (автодополнение кода, чат, рефакторинг) на компьютере с Linux. После установки работа возможна без доступа к интернету.

Требования к системе:
- ОЗУ: от 8 ГБ (рекомендуется 16 ГБ для моделей 7B)
- Диск: от 8 ГБ свободного места
- Интернет: необходим только на время установки и загрузки модели

ШАГ 1 – Установка Ollama (сервер для запуска моделей)
1. Откройте терминал (Ctrl+Alt+T).
2. Выполните команду:
   curl -fsSL https://ollama.com/install.sh | sh
3. После завершения скрипта Ollama автоматически добавится в системные службы.
4. Проверьте установку:
   ollama --version
   (должна показаться версия, например, ollama version 0.3.x)

ШАГ 2 – Загрузка легковесной модели для слабого ПК
В терминале выполните:
   ollama pull qwen2.5-coder:1.5b
Эта модель весит около 3 ГБ и хорошо работает на CPU.
Для проверки запустите чат:
   ollama run qwen2.5-coder:1.5b
(завершить чат – введите /bye)

ШАГ 3 – Установка расширения Continue в VS Code
1. Запустите Visual Studio Code.
2. Нажмите Ctrl+Shift+X (откроется вкладка расширений).
3. В поиске введите Continue и установите расширение.

ШАГ 4 – Настройка Continue для работы с Ollama
1. В VS Code нажмите Ctrl+Shift+P, введите команду:
   Continue: Open Config
2. Откроется файл config.json. Замените его содержимое на:

{
  "models": [
    {
      "title": "Qwen Coder 1.5B",
      "provider": "ollama",
      "model": "qwen2.5-coder:1.5b",
      "apiBase": "http://localhost:11434"
    }
  ],
  "tabAutocompleteModel": {
    "title": "Qwen Coder 1.5B (Autocomplete)",
    "provider": "ollama",
    "model": "qwen2.5-coder:1.5b",
    "apiBase": "http://localhost:11434"
  }
}

3. Сохраните файл (Ctrl+S) и перезагрузите VS Code.

ШАГ 5 – Проверка работы
- На левой панели VS Code появится иконка Continue. Нажмите её – откроется чат. Задайте любой вопрос, например: "Как вывести Hello World на Python?"
- Начните писать код в редакторе – через секунду появится серая подсказка. Нажмите Tab, чтобы принять её.

Всё готово! Теперь вы можете работать полностью офлайн.

Примечания:
- Если модель работает медленно, попробуйте более лёгкую модель: ollama pull deepseek-coder:1.3b и укажите её имя в config.json.
- Чтобы автоматически запускать Ollama при старте системы, выполните: sudo systemctl enable ollama
