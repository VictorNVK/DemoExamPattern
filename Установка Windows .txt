ИНСТРУКЦИЯ ПО УСТАНОВКЕ: Continue + Ollama на Windows

Данная инструкция поможет развернуть полностью локального AI-ассистента для разработки (автодополнение кода, чат, рефакторинг) на компьютере с Windows. После установки работа возможна без доступа к интернету.

Требования к системе:
- ОЗУ: от 8 ГБ (рекомендуется 16 ГБ для моделей 7B)
- Диск: от 8 ГБ свободного места
- Интернет: необходим только на время установки и загрузки модели

ШАГ 1 – Установка Ollama (сервер для запуска моделей)
1. Скачайте установщик с официального сайта: https://ollama.com/download (файл OllamaSetup.exe)
2. Запустите скачанный файл и следуйте инструкциям мастера (Next -> Install -> Finish).
3. После установки Ollama автоматически запустится в фоне.
4. Для проверки откройте командную строку (Win+R, введите cmd) и выполните:
   ollama --version
   Если версия не отображается, перезагрузите компьютер.

(Опционально) Смена папки для моделей:
- По умолчании модели сохраняются на диск C:. Чтобы перенести на D: или другой диск:
  • Откройте “Переменные среды” (поиск в Windows).
  • В разделе “Системные переменные” нажмите “Создать”.
  • Имя: OLLAMA_MODELS, значение: D:\OllamaModels (укажите свой путь).
  • Перезагрузите компьютер.

ШАГ 2 – Загрузка легковесной модели для слабого ПК
1. Откройте командную строку (cmd) или PowerShell.
2. Выполните команду:
   ollama pull qwen2.5-coder:1.5b
   Модель весит около 3 ГБ, загрузка может занять несколько минут.
3. Для проверки запустите чат с моделью:
   ollama run qwen2.5-coder:1.5b
   (завершить чат – введите /bye)

ШАГ 3 – Установка расширения Continue в VS Code
1. Запустите Visual Studio Code.
2. Нажмите Ctrl+Shift+X (откроется вкладка расширений).
3. В поиске введите Continue и установите расширение.

ШАГ 4 – Настройка Continue для работы с Ollama
1. В VS Code нажмите Ctrl+Shift+P, введите команду:
   Continue: Open Config
2. Откроется файл config.json. Замените его содержимое на:

{
  "models": [
    {
      "title": "Qwen Coder 1.5B",
      "provider": "ollama",
      "model": "qwen2.5-coder:1.5b",
      "apiBase": "http://localhost:11434"
    }
  ],
  "tabAutocompleteModel": {
    "title": "Qwen Coder 1.5B (Autocomplete)",
    "provider": "ollama",
    "model": "qwen2.5-coder:1.5b",
    "apiBase": "http://localhost:11434"
  }
}

3. Сохраните файл (Ctrl+S) и перезагрузите VS Code.

ШАГ 5 – Проверка работы
- На левой панели VS Code появится иконка Continue. Нажмите её – откроется чат. Задайте любой вопрос, например: "Как вывести Hello World на C++?"
- Начните писать код в редакторе – через секунду появится серая подсказка. Нажмите Tab, чтобы принять её.

Всё готово! Теперь вы можете работать полностью офлайн.

Примечания:
- Если модель работает медленно, попробуйте более лёгкую модель: ollama pull deepseek-coder:1.3b и измените имя модели в config.json.
- Чтобы Ollama запускалась автоматически, в установщике по умолчанию создаётся задача в планировщике. Если нет – переустановите с опцией “Run at startup”.
